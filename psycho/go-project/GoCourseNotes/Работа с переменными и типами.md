(:=) - инициализация и запись в переменную(сам определяет тип переменной)
(=) - просто запись в переменную
так же инициализировать переменную можно с помощью:
var message string; const message string(можно без типа в обоих способах, если сразу присваиваем переменную, и во втором нужно сразу присвоить переменной значение(перезаписать невозможно))
**переменную нельзя просто создать, её нужно использовать иначе будет ошибка**
**можно объявить сразу несколько переменных в одну строку ** (a, b, c := 1, 2, 3), можно опустить переменную(a, _, c = 1, 5, 6)
типы: 
 - string - строка(нулевое значение - пустая строка, записывается в "a")
 - int(разрядность - 8, 16, 32, 64 ) - целочисленные(нулевое значение - 0)
 - uint(unsigned) - только положительные числа
 - float(с такой же разрядностью, как у int) - числа с плавающей запятой(нулевое значение - 0)
 - bool - true or false(нулевое значение - false)
 - byte - хранит байты ascii, принимает число от 0 до 255
 - rune - хранит символы (записывается в 'a')
 - error - ошибка (нулевое значение - nil) 
 - slice - массив с оберткой(нулевое значение - nil)
 - map
