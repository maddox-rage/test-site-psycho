массив - представляет собой последовательность определенной длинны и состоит из элементов одного типа
[4]int{1, 2, 3, 4}
всегда указываем размер, тип и значения 
можно использовать для хранения последовательностей фиксированной длины(uuid - 16 байт - 128 бит)

слайс - некая структура, которая состоит из 3 элементов(ссылка(pointer: (звездочка)array), длина(lenght: int), емкость(capacity:int))

слайс хранит в себе ссылку на первый элемент базового массива в памяти, а последний элемент определяется с помощью длины
при увеличении емкости создаётся новый базовый массив
-проверять слайс на пустоту лучше с помощью длинны (len(list)= =0)
-если известен размер итогового слайса, то всегда нужно аллоцировать память для него(заранее задать нужную емкость )
-если нужно как-то изменить внутренний слайс, а не внешний, то сначала нужно создать копию
-результат функции append() всегда присваивать той же переменной
